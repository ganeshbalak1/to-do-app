# 🗓️ TODO Web App

A full-stack web application to manage appointments using:

- ✅ Node.js + Express (Backend API)
- ✅ MongoDB (Database)
- ✅ HTML, SCSS, JavaScript, jQuery (Frontend)
- ✅ Webpack (for bundling)
- ✅ Bootstrap 5 for UI components

---

## 🚀 Features

- Register & Login
- Add, Edit, Delete Appointments
- User-specific task list
- Responsive design

---

## 🛠️ Tech Stack

| Layer     | Technology                          |
|-----------|-------------------------------------|
| Frontend  | HTML, SCSS, JS, Bootstrap 5, jQuery |
| Backend   | Node.js, Express                    |
| Database  | MongoDB                             |
| Bundler   | Webpack                             |
| Hosting   | Render                              |

---

## 🧑‍💻 How to Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/ganeshbalak1/to-do-app.git
cd to-do-app

# 2. Install dependencies
npm install

# 3. Build frontend
npm run build

# 4. Start the server
npm start

# 5. Open browser at:
http://localhost:7070
